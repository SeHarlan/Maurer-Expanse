How does it work?

Maurer Expanse an exploration of the Maurer Rose, a geometric concept introduced by Peter M. Maurer. The core algorythm is pretty compact and takes just two variables, "d" and "n":

// where theta is a number from 0 to 361
const k = theta * d * (PI / 180);
const r = 200 * sin(n * k);
const x = r * cos(k);
const y = r * sin(k);

The Expanse is made by creating a grid where the x-axis is "d" and the y-axis "n".


Changing the scale changes how many patterns are displayed at once. Careful here, displaying too many can drastically slow down the experience.


Changing the depth changes the amount of change between each row and column.


And that's pretty much it. Nearly infinite possibilities from just a few lines of code. Happy exploring!
